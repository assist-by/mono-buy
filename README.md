# mono-buy
This project is Monolithic Architecture project for trade in binance.

## 개발 시 실행 방법

```bash
go run .
```

## build 하는 법

```bash
go build
```

## Linux 용 build 하는 법

```bash
# 명령 프롬프트(cmd)에서:
GOOS=linux GOARCH=amd64 go build -o myapp

# PowerShell에서:
$env:GOOS="linux"; $env:GOARCH="amd64"; go build -o myapp
```
